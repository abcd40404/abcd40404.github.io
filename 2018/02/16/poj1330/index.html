POJ 1330 Nearest Common Ancestors

Share Comments
  • Tree

Contents

  1. 1. input
  2. 2. 題解
  3. 3. 作法
  4. 4. AC code

題目

input

第一列輸入 T, 代表 T cases
每筆第一列輸入 N (2 <= N <= 10,000)
代表樹有 N 個點
接著輸入 N - 1 條邊
u, v 代表 u 是 v 的父節點
最後一列輸入 qu, qv, 求 qu, qv 的 LCA

題解

找出最近共同祖先

作法

LCA 裸題

AC code

Newer
LCA (Lowest Common Ancestor)
Older
逆序數對 Inversion
