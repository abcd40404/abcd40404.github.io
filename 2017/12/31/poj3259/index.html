題目

input

含多筆測資, 第一列有一正整數代表接下來有幾筆測資
每筆測資開頭有 3 個正整數 1 <= N <= 500, 1 <= M <= 2500, 1 <= W <= 200
分別代表 N 個點, M 條邊, W 條負邊
接著輸入 M 條邊和 W 條負邊

題解

蟲洞是一條單向路徑, 可以回到過去的時間, 所以等於一條負邊
題目問是否能從一個點開始走, 然後走一走之後, 發現時間比開始的時候還早
那也就是圖中含有負環

作法

可以用 Bellman 或 SPFA 來偵測
