題目

input

n 個歷史事件: 2 ≤ n ≤ 20
接下一行是正確答案的事件rank
後面每行都是學生答案的rank

題解

在一個歷史考試中, 學生被要求根據歷史事件發生年代做排序
全部排對可以得到全部分數, 而部分給分規則如下

  1. 根據每個事件, 排在正確位置就得 1 分
  2. 根據每個事件, 只要他的相對順序是正確就得 1 分
    而我們接下來要採用第 2 個規則
    給一串序列, c1…cn, 代表第 i 個事件排的”位置(rank)”
    例如: 3, 1, 2, 則真實順序為 2, 3, 1
    求每個學生的得分

因為我只對 n 做一次輸入就AC了
應該不用做到多筆測資
